# times： 乘法

## 简介
```ts
times(...args:number[]): number
```

## 参数

1、arg1（number）: 乘数

2、arg2（number）: 乘数

3、arg2（number）: 可选。乘数

……

## 返回

（number）: 相加的结果
## 例子
```js
time(2, 2);
// => 4
plus(2, 2, 2);
// => 8
```
