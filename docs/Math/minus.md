# minus：减法

## 简介
```ts
minus(...args: number[]): number
```


## 参数

1、arg1（number）: 减数

2、arg2（number）: 被减数

3、arg2（number）: 可选。被减数

……

## 返回

（number）: 计算结果

## 例子
```js
minus(2, 1)
// => 1
minus(10, 4, 1)
// => 5
```
